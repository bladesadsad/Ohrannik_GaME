Добро пожаловать в desktop приложение для симуляции терминала охранника посещаемости сотрудников организации! Это руководство пользователя поможет вам ознакомиться с основными функциями и возможностями приложения.

1. Установка ПО:
   - Скачайте все файлы с моего репозитория
   - Перекиньте в проект PyCharm (обязательно нужно иметь установленный PyQT6)

2. Вход в приложение:
   - Запустите файл main.py.
   - вылезает окошко в котором надо будет вводить данные о сотрудниках и не только.

3. Основной интерфейс:
   - В окошке вылезли разные функции.
   - В нижней части окошка множество кнопок , каждая из которых отвечает за свою функцию.
   - В верхней части окошка находятся Label в которые нужно вводить данные о сотруднике или ваши данные.

4. Валидация сотрудников:
   - Для учета посещаемости сотрудников введите данные о сотруднике в Label
   - Нажмите на "пройти на работу"
   - В базе данных автоматически впишется время, когда сотрудник прошёл на работу, если он ввёл верные данные
   - Также работает и с выходом с работы

5. Временные пропуска:
    - Введите свои данные и нажмите на "получить временный пропуск"
    - Выберите время, на которое вам дадут пропуск
    - В базе данных вы появитесь со временем получения пропуска
    - Чтобы выйти по временному пропуску введите ваши данные(которые вы вводили для получения пропуска)
    - Нажмите на "Выйти по пропуску"
    - Если вы не смогли выйти, то вы либо ввели неправильные данные, либо пропуск закончился, если он закончился обратитесь к старшему охраннику
6. Талоны на парковку:
    - Если для сотрудника, то введите фио и номер пропуска
    - Нажмите на "Парковочное талон для сотрудника"
    - Мест всего 5, если останутся места от гостей, то сотрудники смогут занять места гостей
    - Для гостей точно также, только нужно вводить только фио
    - Нажмите на "Парковочный талон для гостя"
    - Мест всего 5, если не останутся места, то охранник предложит вам приехать в другой день
7. Выход из приложения:
   - Чтобы выйти из приложения,нажмите на крестик в правом верхнем углу.

Это основное руководство пользователя для desktop приложения по симуляции терминала охранника посещаемости сотрудников организации. Желаем вам успешной и удобной работы с приложением!
